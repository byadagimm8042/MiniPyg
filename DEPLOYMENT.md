# Deploy Your Landing Page to a Live URL

Here are the easiest ways to get your landing page live:

## Option 1: Netlify Drop (Easiest - No Account Needed Initially)

1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop your entire project folder (or just the files) onto the page
3. Your site will be live instantly with a URL like: `random-name-123.netlify.app`
4. You can customize the URL later in the Netlify dashboard

## Option 2: Netlify (With Account)

1. Go to [https://www.netlify.com](https://www.netlify.com) and sign up (free)
2. Click "Add new site" → "Deploy manually"
3. Drag and drop your project folder
4. Your site will be live with a custom URL

## Option 3: Vercel (Free & Fast)

1. Go to [https://vercel.com](https://vercel.com) and sign up (free)
2. Click "Add New Project"
3. Drag and drop your project folder
4. Your site will be live instantly

## Option 4: GitHub Pages (If you have GitHub)

1. Create a new repository on GitHub
2. Upload your files to the repository
3. Go to Settings → Pages
4. Select the main branch and save
5. Your site will be at: `yourusername.github.io/repository-name`

## Option 5: Surge.sh (Command Line)

If you have Node.js installed:
```bash
npx surge
```
Follow the prompts to create an account and deploy.

---

**Recommended**: Use **Netlify Drop** (Option 1) - it's the fastest and requires no signup initially!
